Minimum Requirements (Basic Setup)


CPU: Dual-core (Intel i3 / Ryzen 3 or equivalent)
RAM: 4 GB (8 GB strongly recommended)
Storage: At least 10–20 GB free
OS:
Windows 10/11 (64-bit)
macOS or Linux also fine
Virtualization: Enabled (required for Docker)
